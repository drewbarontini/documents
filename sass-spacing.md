Sass Spacing
============

Use single-line spaces for separating style declarations.

```sass
.element
    // ...

.another-element
    // ...
```
    
Use single-line spaces for nested blocks.

```sass
.element
    // ...
    
    &:hover,
    &:focus
      // ...

    &:last-child
      // ...

    +respond-to()
      // ...
```
